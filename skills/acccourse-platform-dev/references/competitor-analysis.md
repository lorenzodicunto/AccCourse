## Articulate 360

### Visão Geral
- **Preço**: ~$1.749/usuário/ano
- **Público**: Instrutional designers de todos os níveis
- **Destaque**: Facilidade de uso + suíte completa

### Componentes Principais

| Produto | Tipo | Descrição |
|---------|------|-----------|
| **Storyline 360** | Desktop app | Editor avançado, interface tipo PowerPoint, triggers, layers, estados |
| **Rise 360** | Web app | Editor block-based, responsivo, drag-and-drop, rápido |
| **Content Library** | Recurso | 7M+ assets (imagens, personagens, ícones, vídeos) |
| **Review 360** | Recurso | Plataforma de review e feedback para stakeholders |
| **Reach 360** | LMS | LMS integrado para delivery (opcional) |

### Features-Chave

**Quiz Types (25+)**:
- Multiple choice, Multiple response
- True/False
- Fill-in-the-blank
- Matching, Drag-and-drop
- Hotspot
- Sequence (ordering)
- Numeric, Short answer
- Word Bank
- Free-form: Pick One, Pick Many, Drag-and-drop, Text Entry, Hotspot
- Question pools + randomização + negative scoring

**AI Assistant**:
- Gera conteúdo de slides a partir de prompt
- Gera quiz questions automaticamente
- Reescreve tom (formal/informal)
- Resumo de conteúdo
- AI Voiceover (TTS)
- Closed captions automáticas
- Custom animations via AI
- Feedback variations

**Interatividade**:
- Triggers system (quando X acontece, faça Y)
- Slide layers (sobreposições condicionais)
- Variables e condicionais
- Estados de objetos (normal, hover, selected, disabled, visited)
- Motion paths e animações
- Scrolling panels
- 360° image viewer

**Collaboration**:
- Review 360: link de review, comentários por slide, timeline
- Team Slides: slides compartilhados entre projetos
- Version history

**Output**: SCORM 1.2, SCORM 2004, AICC, xAPI, cmi5, HTML5, MP4, Word

**Acessibilidade**: WCAG 2.1 AA, skip navigation, alt text, captions, keyboard nav

## Adobe Captivate 2025

### Visão Geral
- **Preço**: $33.99/mês ou $1.299 licença perpétua
- **Público**: Designers avançados, projetos complexos
- **Destaque**: Simulações de software, VR, AI avançada

### Features-Chave

**AI Capabilities**:
- AI Voice Narration: converte closed captions em áudio multi-idioma
- Seleção de accents, tones, genders
- AI Image Generation: gera imagens para slides
- AI Avatars: personagens animados por AI
- AI Text Polish: reescreve, expande, corrige gramática
- Bulk caption processing

**Simulations**:
- Screen recording inteligente
- Software simulations (Try, Assess, Training modes)
- Capture de múltiplas resoluções
- Highlight boxes automáticos
- Click boxes com feedback

**VR/360°**:
- 360° image/video import
- Hotspots interativos em VR
- Quizzes dentro de ambientes 360°
- Tracking via xAPI

**Interactive Widgets**:
- Accordion widget
- Tab widget
- Web Objects (embed YouTube, charts, docs)
- Resizable hotspots
- Interactive video
- Drag-and-drop
- Slider
- Click to reveal

**Design**:
- Custom templates exportáveis
- Design systems (exportar/importar)
- Header/footer branding blocks
- Adobe Fonts integration
- Responsive por padrão
- Quick Start Projects (templates prontos)

**Assessments**:
- Question pools
- CSV import de perguntas
- Randomização
- Knowledge check vs avaliação formal
- Branching baseado em resposta

**Output**: SCORM 1.2, SCORM 2004, AICC, xAPI, HTML5, MP4, EXE, Word, Adobe Learning Manager

## Outros Concorrentes

### iSpring Suite
- **Preço**: ~$770/ano
- **Diferencial**: Plugin PowerPoint → converte PPT em e-learning
- **Relevante**: PowerPoint import direto, quizzes robustos, simulações de diálogo

### Lectora
- **Preço**: ~$2.500/ano
- **Diferencial**: Acessibilidade nativa extremamente forte, branching avançado

### Elucidat
- **Preço**: Sob consulta (enterprise)
- **Diferencial**: Collaborative authoring, analytics, brand management

### Evolve Authoring
- **Preço**: Sob consulta
- **Diferencial**: 100% web-based, components library, responsive por padrão

### Rise 360 (standalone comparison)
- **Modelo**: Block-based (cards, accordion, timeline, quiz, etc.)
- **Relevante**: Interface similar ao que AccCourse pode ser — blocos modulares, responsivo, rápido

## Oportunidades de Diferenciação

### Onde AccCourse pode SUPERAR os concorrentes:

1. **AI-First por design** — Concorrentes adicionaram AI como addon. AccCourse pode ser construído com AI no core:
   - "Descreva o curso" → slides + quizzes + narração prontos
   - Auto-sugestões de interatividade baseada no conteúdo
   - AI que sugere melhorias de acessibilidade automaticamente

2. **Preço agressivo** — Articulate: $1.749/ano. Captivate: $408/ano. AccCourse pode entrar a $200-500/ano:
   - Stack open-source (Next.js, SQLite)
   - Self-hosted via Coolify (sem custo de cloud)
   - Modelo per-tenant, não per-user

3. **Simplicidade radical** — Rise 360 é popular pela simplicidade. AccCourse pode ser "Rise + Storyline":
   - Interface block-based (já implementado)
   - Sem curva de aprendizado (drag-and-drop visual)
   - Mas com profundidade (quiz scoring, branching)

4. **Brasileiro/LATAM** — Nenhum concorrente é nativo em PT-BR:
   - Interface em português nativo
   - Suporte a regulamentações brasileiras (NR-1, etc.)
   - Compliance LGPD integrado
   - Latência baixa (VPS no Brasil)

5. **Open architecture** — Flexibilidade que players grandes não oferecem:
   - Custom block types via API
   - Webhooks para integrações
   - White-label por tenant
   - API pública para automação

6. **Modern stack** — Next.js 16, React 19, Turbopack:
   - Performance superior
   - SSR/SSG para landing pages
   - Edge functions possíveis
   - Atualizações constantes
