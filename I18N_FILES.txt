USAGE QUICK START
=================

1. Import hook in client component:
   import { useI18n } from '@/lib/i18n';

2. Use in component:
   const { t, locale, setLocale } = useI18n();
   <h1>{t('dashboard.title')}</h1>

3. Add language switcher to header:
   import { LanguageSwitcher } from '@/components/LanguageSwitcher';
   <LanguageSwitcher />

4. Change language programmatically:
   setLocale('en');

KEY INFORMATION
===============

Total Translation Keys: ~180-200 (45-50 per language × 4 languages)
Code Size: ~1 KB gzipped (excluding translations)
Dependencies: React only (no external i18n libraries)
Performance: Instant language switching, no network calls
Persistence: localStorage with 'i18n-locale' key
Default Language: PT-BR
Type Safety: Full TypeScript support
